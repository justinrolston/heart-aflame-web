Heart Aflame
Contributor Guide
Thank you for your heart to serve and contribute to Heart Aflame! This guide will help you create meaningful, biblically-grounded devotions that encourage believers in their daily walk with Christ.

Our Mission

Heart Aflame provides Reformed believers with daily spiritual nourishment through Scripture-based devotions that are both theologically rich and practically accessible. Each devotion should take 2-5 minutes to read and help cultivate consistent spiritual habits while grounding users in biblical truth.

Key Principle: We aim to kindle affection for Christ and transform daily living through deep theological truth delivered in digestible portions.
Admin Application Overview

Heart Aflame uses a robust admin system for content creation and management. Here's what powers the devotional creation process:

Technology Stack

  • Backend: Ruby 3.4.1 with Sinatra framework
  • Database: Firebase Realtime Database (not Firestore)
  • Templates: ERB templates with responsive CSS design
  • Testing: RSpec framework with 102 comprehensive tests
  • Deployment: Fly.io with Docker and GitHub Actions CI/CD

Admin Features

  • Tab-based editor with live preview functionality
  • Author attribution support for guest contributors
  • Enhanced quotation integration (confessional/catechetical references)
  • Additional resources management with URL validation
  • Markdown import with YAML front matter support
  • JSON export functionality for mobile app integration
  • Responsive design optimized for mobile and desktop use
Quality Assurance: The admin system includes comprehensive data validation, character limits, format checking, CSRF protection, and real-time synchronization with the mobile app.
Current JSON Data Structure

The admin system uses the following JSON structure for storing devotional content (updated 2024):

{
  "daily": {
    "YYYY-MM-DD": {
      "dailyDevo": {
        "title": "Devotional Title",
        "author": "Author Name (optional)",
        "scriptureRef": "Bible Reference",
        "scriptureText": "Scripture verse text",
        "reflection": "Main devotional content",
        "quotationText": "Quote from confession/catechism (optional)",
        "quotationRef": "Source reference (optional)",
        "prayerPrompt": "Prayer guidance (optional)",
        "goDeeper": "Additional study suggestions (optional)",
        "additionalLinks": [
          {
            "title": "Link Title",
            "url": "https://example.com"
          }
        ]
      }
    }
  }
}
Migration Note: The admin system gracefully handles both old and new JSON formats during transition. Legacy devotionals continue to work while new devotionals use the enhanced structure with automatic format conversion for mobile app compatibility.
Devotion Structure & Fields

Each devotion follows a consistent format using the admin interface. Here are the required and optional fields:

  • Title (Required)
    A compelling, descriptive title that captures the essence of your devotion
  • Date (Required)
    Selected from the calendar interface - coordinate with leadership for series timing
  • Author (Optional - New)
    Author attribution for guest contributors and content creators
  • Scripture Reference (Required)
    Biblical reference using ESV translation (e.g., "Romans 14:8")
  • Scripture Text (Required)
    The full Scripture passage from the ESV
  • Reflection (Required)
    The main devotional content connecting Scripture to daily life
  • Quotation Text (Optional)
    Quote from Reformed confessions, catechisms, or theological sources (formerly "Confessional Tie-in")
  • Quotation Reference (Optional)
    Specific source reference (e.g., "Westminster Shorter Catechism Q. 1", formerly "Confession Reference")
  • Prayer Prompt (Optional)
    A brief prayer or prayer guide related to the devotion's theme
  • Go Deeper (Optional)
    Additional reflection questions, applications, or theological insights
  • Additional Links (Optional - New)
    Array of related resources with titles and URLs, validated for proper formatting
Key Changes: Scripture questions have been removed from the current structure. The quotation fields have been simplified and renamed for clarity. Author attribution and additional resource links are now supported.
Writing Guidelines

Length & Readability

  • Target: 2-5 minutes reading time (approximately 300-750 words total)
  • Tone: Warm, pastoral, and accessible while maintaining theological depth
  • Audience: Reformed believers with varying levels of theological knowledge

Scripture Selection

  • Use only the ESV translation for all Scripture references
  • Choose passages that offer rich theological content suitable for reflection
  • Consider both well-known and lesser-known verses that reveal Christ
  • Ensure your chosen passage can sustain 2-5 minutes of meaningful reflection

Theological Approach

  • Ground all reflections in biblical truth and Reformed theology
  • Avoid political topics or controversial contemporary issues
  • Focus on Christ-centered applications that transform daily living
  • Balance doctrinal depth with pastoral warmth
Important: While quotation text is optional, it adds significant value when you can authentically connect Scripture to the Westminster Standards, Heidelberg Catechism, Belgic Confession, or other Reformed theological sources.
Sample Devotion
July 12
Christ, Our Only Comfort
Today's Scripture
For if we live, we live to the Lord; and if we die, we die to the Lord. So, whether we live or die, we belong to the Lord.
— Romans 14:8
Reflection
The Christian's comfort is not found in circumstances, but in the unshakeable truth that we belong to Christ. In both life and death, we are His, and He is ours. This belonging is not merely emotional or sentimental—it is a legal reality, purchased by Christ's blood and sealed by His Spirit.
Quotation
That I am not my own, but belong—body and soul, in life and in death—to my faithful Savior, Jesus Christ.
— Heidelberg Catechism Q1
Prayer
Lord, thank You that I am not my own, but belong entirely to You. Help me to live each day knowing this truth.
Go Deeper
Consider how this fundamental truth affects every aspect of your daily life.
Creating Devotional Series

You're encouraged to create series that span 5-10 days, allowing for deeper exploration of themes, books of Scripture, or doctrinal topics.

Series Ideas

  • Seasonal: Advent, Lent, Easter, Thanksgiving themes
  • Doctrinal: The Five Solas, attributes of God, the Gospel
  • Biblical: Walking through a psalm, the Beatitudes, or a short book
  • Practical: Prayer, worship, Christian character, biblical manhood/womanhood
  • Confessional: Working through questions from catechisms
Timeline: Plan to complete your series within 4 weeks of your start date. Coordinate with leadership for scheduling to ensure proper spacing and avoid conflicts.
Practical Tips
Start with Prayer
Ask God to give you wisdom and insight as you study His Word and prepare to serve others through your writing.
Study Deeply First
Spend time in careful study of your chosen passage before writing. Consider the context, original meaning, and theological significance.
Write Conversationally
Imagine you're sharing this truth with a friend over coffee. Be warm and personal while remaining theologically accurate.
End with Application
Always connect the biblical truth to daily life. How should this passage change how we think, feel, or act today?
Read Aloud
Before submitting, read your devotion aloud to check flow and timing. Aim for natural, readable prose.
Keep Christ Central
Every devotion should ultimately point readers to Jesus Christ and His finished work on their behalf.
Markdown Import Format

The admin system supports importing devotional content via Markdown files with YAML front matter. This is especially useful for content creators who prefer to write in their favorite editor.

---
title: "Devotional Title"
date: YYYY-MM-DD
author: "Author Name (optional)"
scripture_reference: "Bible Reference"
confessional_tie_in: "Quotation text (optional)"
confession_reference: "Reference source (optional)"
prayer: "Prayer prompt (optional)"
go_deeper: "Study suggestions (optional)"
additional_links:
  - title: "Resource Title"
    url: "https://example.com"
---

## Scripture Text
[Scripture verse text]

## Reflection
[Main devotional content]
Note: The YAML front matter field names differ slightly from the JSON structure for readability. The admin system automatically converts between formats during import and export.
Using the Admin Interface
  1. Access the calendar to select your desired date(s) for publication
  2. Click "Create Devotion" to open the editor interface
  3. Fill in all required fields (marked with asterisks in the system)
  4. Add optional fields like quotation text and additional links when appropriate
  5. Use the Markdown import feature if you prefer writing in external editors
  6. Preview your devotion using the Preview tab before submitting
  7. Submit for review - leadership will provide feedback if needed
Remember: You can create multiple devotions as part of a series. Plan your series arc in advance and submit devotions in order for the best reader experience. The admin system includes JSON export functionality for mobile app integration.
